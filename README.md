Apa yang Sebenarnya di pelajari? (The Hidden Curriculum)

1. DOM Traversal: Mereka belajar konsep parentElement. Ini logika penting saat nanti belajar React/Vue atau manipulasi data kompleks.

2. State Logic: Memahami if (kolomSekarang === 'col-todo'). Ini dasar algoritma perpindahan status.

3. Satisfaction Loop: Efek visual kartu berpindah memberikan dopamin instan yang membuat mereka merasa produktif.
